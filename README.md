# 📊 Retail Sales Forecasting & Business Intelligence Dashboard

> An end-to-end Machine Learning project that transforms historical retail sales data into actionable business insights through forecasting, anomaly detection, demand segmentation, and an interactive business intelligence dashboard.

---

## 🌐 Live Demo

🚀 **Interactive Dashboard**

# ✨ Key Features

* 📈 Interactive Sales Analytics Dashboard
* 🔮 Multi-model Sales Forecasting
* 🤖 Prophet, SARIMA & XGBoost Forecasting
* 🚨 Weekly Sales Anomaly Detection
* 📦 Product Demand Segmentation using K-Means
* 📊 Executive Business Intelligence Dashboard
* 🎛 Interactive Filters & Forecast Controls
* 🌙 Modern Glassmorphism UI
* 🌐 Fully Deployed on Streamlit Community Cloud

# 🧠 Machine Learning Pipeline

```text
Retail Sales Dataset
        │
        ▼
Exploratory Data Analysis
        │
        ▼
Feature Engineering
        │
        ▼
Forecasting Models
 ├── Prophet
 ├── SARIMA
 └── XGBoost
        │
        ▼
Model Evaluation
(MAE / RMSE)
        │
        ▼
Anomaly Detection
 ├── Isolation Forest
 └── Z-Score Analysis
        │
        ▼
Product Demand Segmentation
(K-Means Clustering)
        │
        ▼
Interactive Business Dashboard
(Streamlit + Plotly)
```
